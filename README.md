# 🎨 AI Image Creator Studio

AI Image Creator Studio is a Streamlit-based web application that generates AI-powered images using Pollinations AI.

## Features

- 🎨 Generate images from text prompts
- 🖌️ Multiple art styles:
  - Photorealistic
  - 3D Render
  - Pixel Style
  - Ghibli
  - Sketch
  - Manga
  - Anime
  - Cartoon
- ✨ Magic Enhance option for better image quality
- 🎲 Surprise Me feature with random creative prompts
- 📏 Adjustable image width and height
- ⬇️ Download generated images as PNG files

## Technologies Used

- Python
- Streamlit
- Requests
- Pollinations AI

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/mirai-assignment-4.git
```

2. Navigate to the project folder

```bash
cd mirai-assignment-4
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
